Get help choosing a nice restaraunt for dinner, good movie for movie night, or some events to go to. 

This front end React app is created to help you find a nice date night using the yelp api and the tmdb api, current looking for an event api or planning to use yelp's event api to find local events for user. 

There is a development demo for testing.
This is used to gain temporary access to cors anywhere (for yelp api) to access restaurant picker.

https://cors-anywhere.herokuapp.com/corsdemo
